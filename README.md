# Modern Business Website

A responsive, modern business website template built with HTML and CSS. Features a clean design with gradient accents, smooth animations, and mobile-first approach.

<img width="1884" height="3362" alt="image" src="https://github.com/user-attachments/assets/dd6a942b-70b8-427c-b8c0-88d2555103ea" />

## Features

- **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern Design** - Clean layout with gradient backgrounds and smooth hover effects
- **Navigation Menu** - Fixed header with smooth scrolling to sections
- **Hero Section** - Eye-catching introduction with call-to-action button
- **Features Grid** - Display services/features in an organized grid layout
- **About Section** - Two-column layout for company information and imagery
- **Contact Form** - User-friendly form for inquiries (styling only, requires backend)
- **Mobile-Optimized** - Hamburger menu ready for mobile devices (can be enhanced with JavaScript)

## Color Scheme

- Primary Gradient: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- Text Color: `#333` (dark gray)
- Background: `#ffffff` (white)
- Accent Background: `#f9f9f9` (light gray)

## Responsive Breakpoints

- Mobile: Up to 768px
- Tablet: 769px - 1024px
- Desktop: 1025px and above

## Customization Guide

### Colors
To change the color scheme, modify the gradient values in the CSS:
```css
background: linear-gradient(135deg, #your-color 0%, #your-color2 100%);
```

### Content
Replace the placeholder text in these sections:
- Logo text
- Hero section heading and paragraph
- Feature cards (icons, titles, descriptions)
- About section text
- Contact form labels

### Images
Replace placeholder backgrounds with actual images:
```html
<div class="about-image">
    <img src="your-image.jpg" alt="Description">
</div>
```

## Getting Started

1. **Clone the repository**
   ```bash

   ```

2. **Navigate to project directory**
   ```bash
   cd modern-business-website
   ```

3. **Open in browser**
   - Simply open the `index.html` file in your web browser
   - Or use a local development server like Live Server in VS Code
